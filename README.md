# 📝 Menubar Todo (메뉴바 투두)

맥북 상단 메뉴바(상태표시줄)에 쏙 들어가는 가볍고 심플한 투두 리스트 앱입니다. 
언제든 메뉴바 아이콘을 클릭해 빠르게 할 일을 기록하고 관리하세요!

## ✨ 주요 기능
* **빠른 접근성:** 맥북 메뉴바에서 클릭 한 번으로 투두 리스트 열기/닫기
* **다크 모드 지원:** 시스템 테마에 맞춘 부드러운 UI 및 수동 테마 전환 기능 (☀️/🌙)
* **데이터 자동 저장:** 앱을 껐다 켜도 작성한 할 일과 설정이 그대로 유지됨
* **자동 실행:** 설정창에서 '로그인 시 자동 실행' 켜기/끄기 지원

---

## 💿 다운로드 및 설치 방법 (일반 사용자용)

1. 우측의 **[Releases](https://github.com/본인아이디/menubar-todo/releases)** 탭에서 최신 버전의 `.dmg` 파일을 다운로드합니다.
2. 다운로드한 `.dmg` 파일을 더블클릭하여 엽니다.
3. 창이 나타나면 **Menubar Todo** 앱 아이콘을 우측의 **Applications(응용 프로그램)** 폴더로 드래그 앤 드롭합니다.
4. Launchpad 또는 응용 프로그램 폴더에서 앱을 실행합니다.

> **⚠️ 주의: 앱이 열리지 않거나 악성 소프트웨어 경고가 뜨는 경우**
> 개인 개발자가 배포한 앱이므로 맥OS의 보안 정책에 의해 실행이 차단될 수 있습니다. 
> * **해결 방법:** 응용 프로그램 폴더에서 앱 아이콘을 **우클릭(또는 Control + 클릭)** 한 후 **'열기'**를 선택하세요. 경고창이 다시 뜨더라도 '열기' 버튼을 눌러 실행할 수 있습니다. (최초 1회만 해주시면 됩니다.)

---

## 💻 빌드 및 실행 방법 (개발자용)

코드를 직접 수정하고 로컬에서 실행해 보려면 아래 과정을 따르세요.

**1. 저장소 클론 및 패키지 설치**

```bash
git clone [https://github.com/jmh0106/Simple-Menubar-Todo](https://github.com/jmh0106/Simple-Menubar-Todo)
cd menubar-todo
npm install
```
**2. 로컬에서 실행하기**

```Bash
npm start
```

**3. 앱 빌드하기 (macOS용 .dmg 생성)**

```Bash
npm run build
```
빌드가 완료되면 `dist` 폴더 내부에 `.dmg` 파일과 `.app` 파일이 생성됩니다.
